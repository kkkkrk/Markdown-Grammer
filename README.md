# Markdown-Grammer
띄어쓰기 두번 후 엔터: 줄바꿈  

Heading  

#: h1  

##: h2  

...  

######: h6  

# H1

## H2

### H3

#### H4

##### H5

###### H6

#, ##: 라인 추가됨  

___(_3개): 라인 추가  

___

**로 감싸기: bold체  

**hello world**  

*로 감싸기: 이탈릭  

*hello world*

  

~~로 감싸기: strikethrough  

~~hello world~~

">": 인용구  

>hello world  

  

*하나, 혹은 -하나 앞에 달기: bullet list  

* hello world

- hello world

  

1.,2.,3.: 숫자 목록  

1. hello world

2. hello world

  
대괄호+소괄호: 링크  

Click [google](https://google.com)
